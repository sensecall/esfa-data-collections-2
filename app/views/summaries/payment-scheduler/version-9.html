Research notes

Agents were aware from the error message that this claim must already have a payment schedule created, however there was some confusion surrounding whether this was due to the validation issue created for BPS-181.

Agents presumed that the claim must be a duplicate so should therefore be disallowed even in cases where a genuine second claim was provided. This may be because agents have been primarily processing Bereavement Benefit cases so the knowledge surrounding BSP legislation is not fully ingrained. The guidance sheet provided detailing what should be done if a duplicate is suspected caused some confusion as this lead agents to believe 'BSP Manager' to be a button to use under these circumstances rather than the name of the system in which the search should be performed.
